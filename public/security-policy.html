Tuner Security Policy

Overview

Tuner is committed to providing a secure streaming platform for radio and TV content. This policy outlines our security practices and guidelines for safe usage.

Content Security

  • All media streams are delivered over secure HTTPS connections
  • Content is sourced only from verified and legitimate providers
  • No unauthorized or pirated content is permitted
  • Regular security audits are performed on all content sources

Data Protection

  • User data is encrypted in transit and at rest
  • No sensitive personal information is collected or stored
  • Cookies are used only for essential functionality
  • No third-party tracking or analytics scripts are used

Technical Security Measures

  • Strict Content Security Policy (CSP) implementation
  • Regular security updates and vulnerability patching
  • HTTPS-only with HSTS enabled
  • Protection against XSS, CSRF, and other common web vulnerabilities

Reporting Security Issues

If you discover a security vulnerability, please report it to security@tuner.live. We appreciate responsible disclosure and will respond promptly to all security reports.

Updates

This security policy was last updated on April 15, 2024. We regularly review and update our security measures to maintain the highest standards of protection.
